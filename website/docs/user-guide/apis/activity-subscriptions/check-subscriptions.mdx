# Check activity subscriptions

import GoExample from '@site/src/components/GoExample';
import asgGo from '@site/snippets/actsub_guide.go';

List activity-stream activities and check whether the current user is subscribed to an object.

## When to use this pattern

- You're building notification or follow/unfollow features on top of ServiceNow's activity streams
- You need to know what the integration user is already subscribed to

## Required values

| Value | Description |
| ----- | ----------- |
| Object ID | The sys_id of the object whose subscription state you're checking |
