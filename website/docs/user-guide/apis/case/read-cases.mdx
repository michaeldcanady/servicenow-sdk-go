# List and read CSM cases

import GoExample from '@site/src/components/GoExample';

List Customer Service Management cases and read their fields.

## When to use this pattern

- You're syncing CSM cases into an external ticketing or reporting system
- You're building customer-facing status views on top of ServiceNow cases

## Required values

| Value | Description |
| ----- | ----------- |
| Case filters | Optional query parameters narrowing which cases you get |

## Tips

- Case models are typed — `GetNumber()`, `GetShortDescription()`, `GetState()`, `GetPriority()` — no generic field unwrapping needed.
- `Post` creates a case; `ByID(...).Put` updates one.
