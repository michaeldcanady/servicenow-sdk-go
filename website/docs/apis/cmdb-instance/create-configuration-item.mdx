# Create configuration item

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Create a CI in a CMDB class.

## HTTP request

```http
POST /api/now/v1/cmdb/instance/{className}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `className` | Classname |

## Query parameters

None.

## Request body

Supply a `cmdbinstanceapi.CmdbInstance`.

## Response

An item envelope of the created `CmdbInstance`.
