# Get configuration item

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve a CI by class and sys_id.

## HTTP request

```http
GET /api/now/v1/cmdb/instance/{className}/{sys_id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `className` | Class name |
| `sys_id` | Sys_id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

An item envelope of `CmdbInstance`.
