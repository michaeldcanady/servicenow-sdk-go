# Patch configuration item

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Partially update a CI's attributes.

## HTTP request

```http
PATCH /api/now/v1/cmdb/instance/{className}/{sys_id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `className` | Classname |
| `sys_id` | Sys_id |

## Query parameters

None.

## Request body

Supply a `cmdbinstanceapi.CmdbInstance`.

## Response

An item envelope of the updated `CmdbInstance`.
