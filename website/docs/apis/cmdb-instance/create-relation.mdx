# Create CI relation

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Create a relationship between CIs.

## HTTP request

```http
POST /api/now/v1/cmdb/instance/{className}/{sys_id}/relation
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `className` | Class name |
| `sys_id` | Sys_id |

## Query parameters

None.

## Request body

Supply a `cmdbinstanceapi.CmdbInstance`.

## Response

An item envelope of `CmdbInstance`.
