# Get changeset's impacted deployables

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve the impacted deployables for a specific changeset.

## HTTP request

```http
GET /api/sn_cdm/changesets/{changeset_id}/impacted-deployables
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `changeset_id` | Changeset id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`ImpactedDeployablesBySysIdResponse`.
