# List changesets

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve configuration-data changesets.

## HTTP request

```http
GET /api/sn_cdm/changesets
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `appName` | Optional — set via the verb's `QueryParameters` type. |
| `number` | Optional — set via the verb's `QueryParameters` type. |
| `state` | Optional — set via the verb's `QueryParameters` type. |
| `changesetNumber` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Don't supply a request body for this method.

## Response

`ChangesetsResponse` — a collection envelope of `*ChangesetResult`.
