# Get commit status

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Check the status of a changeset commit.

## HTTP request

```http
GET /api/sn_cdm/changesets/commit-status/{commit_id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `commit_id` | Commit id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`CommitStatusResponse`.
