# Subscribe to object

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Subscribe the current user to an object.

## HTTP request

```http
POST /api/now/v1/actsub/subscriptions/{sub_obj_id}/subscribe
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `sub_obj_id` | Sub obj id |

## Query parameters

None.

## Request body

Supply a `actsubapi.ActivitySubscriptionModel`.

## Response

An item envelope of `*ActivitySubscriptionModel`.
