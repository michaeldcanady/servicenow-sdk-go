# Get subscribers

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve the subscribers for an object.

## HTTP request

```http
GET /api/now/v1/actsub/subscribers/{sub_object}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `sub_object` | Sub object |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

A collection envelope of `*ActivitySubscriptionModel`.
