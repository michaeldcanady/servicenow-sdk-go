# Create case

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Create a CSM case.

## HTTP request

```http
POST /api/sn_customerservice/v1/case
```

## Path parameters

None.

## Query parameters

None.

## Request body

Supply a `caseapi.CaseResult`.

## Response

`CaseItemResponse` — an item envelope of the created case.
