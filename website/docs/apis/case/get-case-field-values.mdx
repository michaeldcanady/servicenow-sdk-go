# Get case field values

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve the allowed values for a case field.

## HTTP request

```http
GET /api/sn_customerservice/v1/case/field_values/{field_name}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `field_name` | Field name |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`FieldValuesResponse`.
