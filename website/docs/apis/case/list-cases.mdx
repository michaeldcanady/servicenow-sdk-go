# List cases

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve CSM cases, optionally filtered by an encoded query.

## HTTP request

```http
GET /api/sn_customerservice/v1/case
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `sysparm_query` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Don't supply a request body for this method.

## Response

`CaseCollectionResponse` — a collection envelope of `*CaseResultModel`.
