# Update case

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Update a CSM case by sys_id.

## HTTP request

```http
PUT /api/sn_customerservice/v1/case/{id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `id` | Id |

## Query parameters

None.

## Request body

Supply a `caseapi.CaseResult`.

## Response

`CaseItemResponse` — the updated case.
