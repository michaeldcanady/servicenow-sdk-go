# Get case

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve a CSM case by sys_id.

## HTTP request

```http
GET /api/sn_customerservice/v1/case/{id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `id` | Id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`CaseItemResponse`.
