# List nodes

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve configuration data nodes.

## HTTP request

```http
GET /api/sn_cdm/editor/v1/nodes
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `sys_id` | Optional — set via the verb's `QueryParameters` type. |
| `parent_id` | Optional — set via the verb's `QueryParameters` type. |
| `type` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Don't supply a request body for this method.

## Response

`NodesResponse`.
