# Delete node

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Delete a configuration data node.

## HTTP request

```http
DELETE /api/sn_cdm/editor/v1/nodes/{node_sys_id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `node_sys_id` | Node sys_id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`NodeDeleteResponse` — deletion outcome details.
