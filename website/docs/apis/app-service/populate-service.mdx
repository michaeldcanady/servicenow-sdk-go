# Populate service

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Populate an application service's topology.

## HTTP request

```http
PUT /api/now/v1/cmdb/csdm/app_service/{sys_id}/populate_service
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `sys_id` | Sys_id |

## Query parameters

None.

## Request body

Supply a `appserviceapi.PopulateServiceRequest`.

## Response

`PopulateServiceResponse`.
