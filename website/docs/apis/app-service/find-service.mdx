# Find service

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Find a CSDM application service by name or number.

## HTTP request

```http
GET /api/now/v1/cmdb/csdm/app_service/find_service
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `name` | Optional — set via the verb's `QueryParameters` type. |
| `number` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Don't supply a request body for this method.

## Response

`FindServiceResponse`.
