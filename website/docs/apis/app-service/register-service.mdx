# Register service

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Register an existing application service in the CSDM model.

## HTTP request

```http
POST /api/now/v1/cmdb/csdm/app_service/register_service
```

## Path parameters

None.

## Query parameters

None.

## Request body

Supply a `appserviceapi.RegisterServiceRequest`.

## Response

`RegisterServiceResponse`.
