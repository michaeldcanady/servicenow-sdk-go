# Update deployables

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Update a deployable's configuration data.

## HTTP request

```http
PUT /api/sn_cdm/applications/deployables
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `appName` | Optional — set via the verb's `QueryParameters` type. |
| `name` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Supply a `cdmapplicationsapi.DeployableUpdateRequest`.

## Response

`DeployableUpdateResponse`.
