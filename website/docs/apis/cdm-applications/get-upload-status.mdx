# Get upload status

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Check the status of an asynchronous configuration upload.

## HTTP request

```http
GET /api/sn_cdm/applications/upload-status/{upload_id}
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `upload_id` | Upload id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`UploadStatusResponse`.
