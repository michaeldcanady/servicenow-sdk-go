# Get export content

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Download a completed export's content.

## HTTP request

```http
GET /api/sn_cdm/applications/deployables/exports/{export_id}/content
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `export_id` | Export id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`[]byte` — the raw export content.
