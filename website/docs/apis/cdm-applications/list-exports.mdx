# List exports

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Retrieve a deployable's configuration exports.

## HTTP request

```http
GET /api/sn_cdm/applications/deployables/exports
```

## Path parameters

None.

## Query parameters

| Name | Description |
| ---- | ----------- |
| `appName` | Optional — set via the verb's `QueryParameters` type. |
| `deployableName` | Optional — set via the verb's `QueryParameters` type. |

## Request body

Don't supply a request body for this method.

## Response

`ExportsResponse`.
