# Get export status

import GoSnippet from '@site/src/components/GoSnippet';
import refGo from '@site/snippets/reference_actions.go';

Check the status of a configuration export.

## HTTP request

```http
GET /api/sn_cdm/applications/deployables/exports/{export_id}/status
```

## Path parameters

| Name | Description |
| ---- | ----------- |
| `export_id` | Export id |

## Query parameters

None.

## Request body

Don't supply a request body for this method.

## Response

`ExportStatusResponse`.
